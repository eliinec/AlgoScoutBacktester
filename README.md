# AlgoScoutBacktester
A Python-based algorithmic trading backtester using Pandas, Scikit-Learn, and Matplotlib to evaluate Moving Average strategies.

## Features: 
Dual Moving Average Crossover (5day vs 20day)
Usses Matplotlib charts for performance and visualization 
Calculates Maximum drawdowns and returns
Uses Scikit-Learn for price trends

## Installation: 
1. Clone the repository
2. Install: pip install -r requirements.txt
3. Run python algo_scout.py
